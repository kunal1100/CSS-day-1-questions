CSS INTERVIEW QUESTIONS DAY 1.


1. Difference between flex row and flex column

Flex row = works in horizontl direction, childs in div placed into a horizontal way

Flex column = works in vertical direction, childs in div placed into a vertical way



2. Difference between inline CSS and external CSS.

Inline CSS = Inline CSS can be used as an attribute using style tag

External CSS = External CSS can be used by creating a new file and linking it to the html file



3. What is justify contents.

Justify content allows us to distribute items in X axis or Y axis, justify content used with flex boxes



4. Absolute and relative position difference.

Relative position - Its is used when we have to put the element into a normal position.

Absolute position - It will work according to the parent container or body can if the element does not have a parent container.



5. What is grid template column.

Grid template column is a sub property of grid, and it is used to give columns to a page, it takes values in pixels, fr etc.



6. Difference between padding and margin.

Padding = Applied inside of the border, you should not give negative value to padding.

Margin - Applied outside o the border, Margin is ok with both +ve and -ve values.



7. Explain z-index

Z-index property is used to define the stack level of the element Elements having higher index value will be placed on the top of the other elements which is having a lower value. It takes values in negative as well as positive. Z-index will only works on an element whose position is set to either absolute, relative or fixed.
